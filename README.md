Lista de tarefas para o programa de treinamento.

## Descrição

 Este é um projeto simples com 10 exercícios projetados para ajudar você a aprimorar suas habilidades em C# durante esta semana de aprendizado intensivo. 
 Cada exercício aborda conceitos fundamentais da linguagem, proporcionando uma base sólida para o desenvolvimento em C#.
